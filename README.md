# CMSC-313-Project-3

Project3.c contains the C code for the hexdump. The files Opt0.s, Opt1.s, and Opt3.s contain the assembly code, each with increased optimizations. Opt0.s contains no optimizations and is a direct translation of the C code. The code in this file is much easier to read and debug compared to the rest but in turn it is the most inefficient in speed. Op1.s has moderate improvements which increase its speed but maintains the readability. Opt3.s is the file with the most optimized code. The code in this file is a bit harder to follow, but it is the fastest.
